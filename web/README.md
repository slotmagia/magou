# HotGo 2.0 前端项目

这是 HotGo 2.0 的前端项目，基于 Vue 3 + TypeScript + Naive UI 构建的现代化管理系统。

## 🚀 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev

# 构建生产版本
pnpm run build
```
